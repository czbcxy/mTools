# 安装到访达工具栏（核心用法）

mTools 的正确用法是**把按钮装到访达工具栏上**，点一下就在当前访达窗口里干活，不弹独立窗口。

## 三个按钮

| 按钮 | 作用 | 点击后效果 |
|------|------|-----------|
| **Open Terminal Here** | 在当前目录打开终端 | 直接打开 Terminal 并 cd 到访达当前位置 |
| **Copy Path** | 复制当前路径 | 把访达当前路径复制到剪贴板（顶部通知提示） |
| **Show Path Bar** | 显示/隐藏路径栏 | 切换访达底部的路径栏，点里面的文件夹即可逐级跳转 |

面包屑上下级跳转 = 用 **Show Path Bar** 打开访达原生路径栏，点路径栏里的文件夹就是跳转（macOS 原生"面包屑"）。

## 安装步骤

**从 dmg 安装包（推荐）：**
1. 双击 `dist/mTools.dmg`
2. 把三个 `.app` 拖到访达窗口里的 **Applications**
3. 打开任意访达窗口，按住 ⌘ 把 Applications 中的三个 `.app` 逐个拖进访达工具栏右上角
4. 终端执行 `killall Finder` 刷新

**从源码 `apps/` 目录：**
1. 打开访达，进入项目里的 `apps/` 目录（本机路径 `/Users/chengzhengbo/WorkBuddy/super-addr/apps`；或按 ⇧⌘G 前往）
2. 按住 ⌘ 把三个 `.app` 逐个拖进访达工具栏右上角

> 想整理图标顺序：按住 ⌘ 拖动工具栏图标即可重排；拖出工具栏即删除。
> 建议先把 .app 放到 Applications 固定位置，工具栏按钮引用它才不会在 dmg 弹出后失效。

## 首次授权

点按钮时访达需要被控制，按系统提示在
**系统设置 → 隐私与安全性 → 辅助功能**（或 自动化）
里把 **Terminal**（以及运行这些 .app 的程序）加上。

## 使用

- 在访达里浏览到某个目录，点工具栏的 **Copy Path** → 路径已复制
- 点 **Open Terminal Here** → 终端在该目录打开
- 点 **Show Path Bar** → 底部出现路径栏，点其中任意文件夹直接跳转层级

全程不离开访达，没有额外窗口。
